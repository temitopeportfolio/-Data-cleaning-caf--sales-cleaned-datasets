# cafe--sales-cleaned-datasets
This project contains a cleaned and structured cafe sales datasets prepared for data anaysis and visualization.

## Project Overview
The raw was cleaned to improve data quality, constitency, and usuability for analysis and reporting purposes.

## Data Cleaning Task Perfromed
- Handled missing values
- Corrected inconstitent data enteries
- Fixed data and numeric formats
- Standardized column formatting
- Organized datasets for analysis

  ## Tools Used
  - Microsoft Excel
  - Power query
 
  ## Purpose
  - This project was created to practice and demonstaratedata cleaning skills as part of of my data analytics learning journey.
